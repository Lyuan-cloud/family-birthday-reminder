# Family Birthday Reminder Tool

A web application to help families remember and celebrate birthdays together.

## Features

- Birthday tracking and reminders
- Email notifications
- Quote wall for sharing birthday wishes
- Image upload for birthday cards

## Tech Stack

- Frontend: Next.js + React
- Backend: Cloudflare Workers
- Database: Cloudflare D1
- Storage: Cloudflare R2
